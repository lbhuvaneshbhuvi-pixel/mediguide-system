# **App Name**: MediGuide

## Core Features:

- Symptom Input: Allow users to input symptoms via text or voice in English or Tamil.
- AI-Powered Disease Prediction: Use an ML/NLP model to predict possible diseases/conditions based on user-provided symptoms and recommend ONE most suitable medicine.
- Medicine Information Display: Show medicine details including dosage, side effects, precautions, and alternatives.
- Bilingual Support: Enable the entire application to switch between English and Tamil, including AI-generated responses and voice output. The AI model uses a tool for deciding when the response must include information about medicine alternatives.
- Voice Interaction: Implement Text-to-Speech (TTS) and Speech-to-Text (STT) functionality for voice input and output in both languages.
- Database Storage: Store symptom-disease mappings, disease-medicine mappings, and medicine details including manufacturer, dosage, precautions, and alternatives.

## Style Guidelines:

- Primary color: Light Blue (#B0E2FF), providing a calm and trustworthy feel.
- Background color: Very light cyan (#F0FFFF), near-white and unobtrusive to ensure readability.
- Accent color: Deep sky blue (#00BFFF), a more vibrant blue tone for interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) to give a clean, modern, and accessible feel.
- Use clear and recognizable icons for medicine types and symptoms.
- Design a clean, modern, minimal, and easy-to-use interface with a prominent search bar and voice input button on the landing page.
- Subtle animations on AI prediction loading and language toggling to enhance user experience.
